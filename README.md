## 介紹
- 登入登出切換帳號
- 登入後可以新增刪除喜歡的餐廳

### 功能
- 註冊登入登出功能
- 瀏覽一家餐廳的詳細資訊
- 瀏覽全部所有餐廳
- 搜尋餐廳

### 開始使用
- 安裝node.js and npm
- 複製專案到本機
- npm install
- npm run seed >> 種子資料
- npm run dev

看到以下訊息表示專案正常運行
- express is running on localhost:3000
- mongodb connected 

- 連線 localhost:3000 進入專案
